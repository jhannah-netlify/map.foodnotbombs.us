OFFICIAL MAP (hundreds of groups, but out of date)

To add your group to the DRAFT UNOFFICIAL map above:

  • Open a pull request against our groups.js file.
  • If you don't know what a pull request is, that's fine, create a github Issue with all your group information and we'll add it for you.
  • If you don't have or want a github account that's fine, just email your group info to jay@jays.net and he'll add it for you.
